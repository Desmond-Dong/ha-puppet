# 更新日志

## v1.20.0 (2025-11-13)

### 🎨 黑白墨水屏转换优化

- **改进的字体渲染**:
  - 优化字体回退链，确保中文显示清晰
  - 启用抗锯齿和更好的文本渲染
  - 强制禁用可能导致字体透明或模糊的CSS属性

- **新的图像处理算法**:
  - 实现 **Floyd-Steinberg 抖动算法** 替代简单阈值
  - 添加智能对比度增强
  - 轻微锐化处理提高清晰度
  - 大幅减少黑白转换时的锯齿感

- **依赖更新**:
  - Puppeteer: 24.2.1 → 24.6.1 (性能和稳定性改进)
  - Sharp: 0.34.1 → 0.35.0 (图像处理性能优化)

### 🐛 修复的问题

- 修复黑白转换时字体模糊不清晰的问题
- 修复图像边缘锯齿严重的问题
- 改善在墨水屏上的可读性

### 🔧 技术改进

- 使用 Floyd-Steinberg 抖动实现更平滑的灰度到黑白转换
- 优化字体渲染，确保在各种屏幕尺寸下都清晰可读
- 改进图像锐化算法，平衡清晰度和锯齿
